# Portfolio Recruiter-First

Ce dossier contient une base de site portfolio orientée recruteurs.

## Fichiers
- `index.html`: structure du site.
- `styles.css`: design responsive.
- `script.js`: menu mobile + animations d'apparition.

## Lancer localement
1. Ouvre `portfolio/index.html` dans ton navigateur.

## Personnalisation rapide
1. Remplace `Prénom Nom` et les infos de contact.
2. Mets ton vrai lien LinkedIn/GitHub.
3. Remplis les sections Expériences et Projets avec des résultats chiffrés.
4. Ajoute ton CV PDF et remplace le lien `Télécharger le CV (PDF)`.
